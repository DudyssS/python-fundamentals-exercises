## Objetivo

Este repositório tem como objetivo consolidar fundamentos da linguagem Python através de exercícios práticos, desenvolvendo lógica de programação, raciocínio matemático e estruturação de código.

---

## 🧠 Estrutura dos exercícios

Os arquivos estão organizados por tipo de problema:

---

### Matemática e Cálculos

- `basic_arithmetic_operations.py` → operações básicas
- `quadratic-equation-solver.py` → equação do 2º grau
- `temperature_converter.py` → conversor de temperatura
- `coin-change-calculator.py` → cálculo de troco
- `binary_to_decimal.py` → conversão binário/decimal
- `RandomNumber.gs` → geração de números aleatórios

---

### Lógica e Condições

- `logic_conditions.py` → lógica condicional
- `decision_tree_logic.py` → árvore de decisão

---

### Estruturas de Dados e Loops

- `loops_and_data_structures.py` → loops + estruturas

---

### Strings e Manipulação de Texto

- `string_methods_practice.py` → métodos de string

---

### Sistemas e Simulações

- `simple-login-authentication.py` → sistema de login simples
- `calendar_operations.py` → operações com datas/calendário

---

### Exercícios Extras / Diversos

- `Parsecs to lightyears.py` → conversão científica
- `matrix_and_image_processing.py` → introdução a processamento de matriz/imagem

---
